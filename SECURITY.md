## Reporting a Vulnerability

If you believe you have found a security vulnerability, please report it by emailing our security team at **security@example.com** with the subject line `SECURITY REPORT: <project name>`.

Include as much detail as possible in your report (e.g., affected versions, configuration, reproduction steps, impact, and any proof‑of‑concept code). Please do **not** open a public issue for security‑sensitive information.

We aim to acknowledge all security reports within **72 hours** and provide an initial assessment or follow‑up questions within **5 business days**. You will receive updates as we investigate, validate, and work on a fix.

Once a vulnerability is confirmed, we will:

- Work privately on a fix and any necessary mitigations.
- Prepare a security release and update relevant documentation.
- Coordinate a responsible disclosure timeline with you when appropriate.

After a fix is released, we may publish a security advisory describing the issue, impact, and resolution. If you would like credit for the discovery, please let us know how you wish to be acknowledged.
